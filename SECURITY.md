# Security Policy

## Reporting a Vulnerability

Please report bypasses, false negatives, or unsafe defaults through GitHub security advisories when available, or by opening a private issue with the maintainer.

Do not include live secrets in reports. Use redacted examples or synthetic tokens that match the same shape.

## Scope

Local Secret Leak Preflight is a best-effort local guardrail. It is designed to catch common accidental leaks before commit, not to replace secret rotation, repository secret scanning, or production monitoring.
